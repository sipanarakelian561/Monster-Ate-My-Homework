# Monster Ate My Homework (C# + Godot)

An educational game built with **Godot** and **C#**.

## Tech Stack

- Godot (C#/.NET)
- C# scripts for gameplay, questions, and UI
- (Planned) SQL databases for Math, English, and Science question sets

## How to Run

1. Open Godot.
2. Choose **Import Project**.
3. Select `project.godot` in this folder.
4. Click **Run**.

## Project Goals

- Help K–6 students practice Math, English, and Science.
- Solve problem sets, then unlock a mini-game ("Monster Ate My Homework").
